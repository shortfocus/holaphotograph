🔥 섹션별 구체 개선안

1️⃣ 전문가 소개 섹션

문제
• 왼쪽 아이콘이 너무 UI 느낌
• 정보 밀도가 낮아서 허전
• 다음 섹션과 시각적 단절이 없음

해결

✅ 아이콘 제거 → 실제 프로필 사진 사용
✅ 배경을 아주 연한 그레이 블록으로
✅ 위에 구분선 or padding 더 크게

그리고 타이틀 아래 설명 추가:
실사용 기반 장비 분석과 촬영 노하우를 공유합니다.

지금은 제목만 있고 맥락이 부족해.

---

2️⃣ 실전 영상 리뷰

문제
• 타이틀과 카드 사이 간격이 애매
• 섹션 설명 없음
• 다음 섹션과 배경 톤 차이 약함

해결

타이틀 구조 이렇게 바꿔:
📺 실전 영상 리뷰
직접 사용하고 비교한 장비 리뷰 영상

그리고 이 섹션은 흰 배경 유지
→ 다음 섹션은 회색 배경으로 강하게 분리

---

3️⃣ 빠르게 보는 장비 팁 (Shorts)

문제
• 위 섹션과 구분 약함
• 카드가 많아서 시각적으로 복잡

해결

✅ 배경을 연한 그레이로 변경
✅ 타이틀 아래 설명 추가:
1분 안에 핵심만 정리한 장비 팁

그리고 카드 gap 조금 늘려.

지금은 살짝 붙어 보임.

---

4️⃣ 장비 비교 & 상세 분석 (블로그)

문제
• 3열 카드가 많아서 덩어리로 보임
• 위 섹션과 구조가 비슷해서 반복 느낌

해결

이 섹션은 흰 배경 유지.

타이틀 아래 설명:
구매 전 꼭 확인해야 할 비교 분석 콘텐츠

그리고 3열 유지하되 카드 높이 완전 통일.

---

5️⃣ 숫자 성과 섹션

문제
• 너무 조용함
• 위 블로그 섹션과 시각적 연결 약함

해결

이 섹션을 회색 배경 블록 안에 넣어.

위에 작은 라벨 추가:
누적 성과

숫자는 조금 더 키워.

---

6️⃣ 강의 티저

이건 좋다.
근데 위 숫자 섹션과 구분이 애매함.

해결:
• 숫자 섹션은 밝은 톤
• 강의 섹션은 네이비

강한 대비 주면 구분 명확해짐.
