# 올라포토 웹사이트 업데이트 안내

**작업 완료일**: 2025년 2월

---

## 1. 변경 사항 요약

### 네이버 블로그 아이콘
- 최신 네이버 블로그 로고로 교체
- 마우스를 올리면 색상이 바뀌는 효과 적용

### 전문가 소개 섹션
- 해당 섹션 제거 (페이지 구성 간소화)

### 장비 비교 섹션
- **변경 전**: 카드가 2~3열로 나열
- **변경 후**: 가로로 스와이프하며 볼 수 있게 변경 (Shorts와 동일한 형태)

### 강의 소식 받기
- **변경 전**: 페이지 중간에 한 번만 노출
- **변경 후**: 스크롤해도 화면 하단에 항상 고정되어 보임

---

## 2. 콘텐츠 노출 기준

### 유튜브 영상

| 섹션 | 노출 기준 | 최대 개수 |
|------|-----------|-----------|
| **실전 영상 리뷰** | 1분 이상 영상, 제목에 리뷰·언박싱·비교·사용기·후기 등 포함 시 우선 | 6개 |
| **빠르게 보는 장비 팁** | 1분 이하 영상, 제목에 [대여] 있으면 제외, 팁·꿀팁·핵심 등 포함 시 우선 | 6개 |

### 네이버 블로그

| 섹션 | 노출 기준 | 최대 개수 |
|------|-----------|-----------|
| **장비 비교 & 상세 분석** | 제목 또는 카테고리에 "렌즈 리뷰" 또는 "카메라 리뷰" 포함 | 6개 |

---

## 3. 참고 사항

- 유튜브 영상은 채널에 올라온 순서(최신순)를 기준으로 가져옵니다.
- 블로그 글은 제목·카테고리 키워드로 자동 분류됩니다.
- 노출 개수는 각 섹션당 최대 6개로 제한됩니다.

---

## 4. 유튜브 검색 API 활용 방안

현재는 **채널 전체 영상**을 가져와 재생 시간·제목으로 분류하고 있습니다.  
유튜브 검색 API를 활용하면 아래와 같은 방식으로 콘텐츠를 더 세밀하게 구성할 수 있습니다.

| 방안 | 설명 | 활용 예시 |
|------|------|-----------|
| **키워드 검색** | 특정 검색어로 관련 영상만 가져오기 | "카메라 입문", "렌즈 비교" 등 키워드별 섹션 구성 |
| **인기순 노출** | 조회수·인기도 기준으로 정렬 | 인기 영상 섹션, 베스트 리뷰 코너 |
| **주제별 섹션** | 검색어마다 다른 섹션에 노출 | 입문 가이드 / 장비 비교 / 촬영 팁 등 주제별 분리 |
| **다중 채널** | 여러 채널의 영상을 한 화면에 | 협업 채널·제휴 콘텐츠 통합 노출 |

**도입 시 고려 사항**
- **일일 할당량**: 유튜브 API는 무료로 하루 10,000 유닛이 제공됩니다. 현재 사이트 규모와 방문자 수를 고려하면 한도를 넘을 가능성은 낮습니다. 다만 검색 키워드를 많이 추가하거나 트래픽이 크게 늘어날 경우를 대비해, 결과를 일정 시간 저장해 재사용하는 방식으로 안전하게 운영할 수 있습니다.
- 검색 키워드 설계에 따라 노출 품질이 달라짐
