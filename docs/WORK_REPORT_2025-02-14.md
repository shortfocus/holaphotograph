# 오늘 작업 내역 (고객 공유용)

**작업일:** 2025년 2월 14일  
**대상:** 홀라포토(holaphoto.com)

---

## 1. 메인 페이지 강의 홍보 팝업 UI

- 메인 진입 시 1회 노출 팝업
- 강의 안내 슬라이드 2장(노출 3요소 / X-E5 클래스), 공지 상세(/notice?id=2) 링크
- **오늘 하루 안 보기** 체크 시 로컬스토리지에 저장해 당일 재노출 방지
- 닫기 버튼·백드롭 클릭 시 닫기

---

## 2. 공지사항 기능

- **공지사항 게시판:** 메인 사이트 공지 목록·상세(모달), 기존 공지 수신 신청 폼 유지
- **관리자 공지 관리:** 목록·등록·수정·삭제, Trix 에디터·이미지 업로드, 본문 이미지 URL 처리(메인/관리자 도메인 공통)

---

## 3. 백엔드

- **DB:** 공지사항 테이블 마이그레이션
- **API:** 공지 조회·관리자 CRUD·이미지 업로드
- **클라이언트:** `api.ts`에 공지 조회·관리자 CRUD·이미지 업로드 함수 추가
